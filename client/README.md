# Truecaller-blog client

- Built with Next.js, Typescript, Axios & Rebass.

### Available commands:

- **start**

  ```bash
  yarn dev
  ```

- **build**

  ```bash
  yarn build
  ```

- **lint**

  ```bash
  yarn run lint
  ```

- **lint fix**

  ```bash
  yarn run lint:fix
  ```

### Pages:

- `/`
- `/?category=foo`
- `/?tag=bar`
- `/post/[postId]`
- `404` & `Internal Error page`
